### 1. Interactive Project Hub (`index.html`)
The central landing page that organizes and links all hosted artifacts. It provides a clean, responsive UI to navigate between different tools and guides.

### 2. Mastering uv (`uv-guide.html`)
An interactive technical guide for **uv**, the extremely fast Python package manager written in Rust.
* **Features:** Interactive terminal simulator, dependency flow diagrams, and a searchable command cheat sheet.
* **Tech Stack:** Tailwind CSS, Chart.js, Vanilla JS.



### 3. Hedging Simulator (`hedging_simulator.html`)
A financial visualization tool used to simulate the **"Stock + Put Option"** strategy.
* **Features:** Real-time sliders to adjust stock price, strike price, and option costs with dynamic Profit/Loss chart overlays.
* **Tech Stack:** Chart.js, HTML5 Canvas.



### 4. Weather WebApp (External)

## ðŸ”§ Deployment
To host these files yourself:

1.  **Clone** this repository to your local machine.
2.  Ensure your `index.html` is in the **root directory**.
3.  Navigate to your repository on GitHub.
4.  Go to **Settings > Pages**.
5.  Set the **Source** to the `main` branch and click **Save**.
